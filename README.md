# Clinical Evidence Platform

A Korean clinical evidence search and research support platform for medical trainees and residents.

## Overview
This project aims to help medical trainees quickly find:
- Korean clinical papers
- case reports
- guidelines
- research references

It also aims to support research workflows such as:
- dataset exploration
- basic statistical analysis
- evidence organization

## Core Goals
- Reduce time spent searching for evidence
- Prioritize Korean clinical relevance
- Separate domestic and international evidence clearly
- Support future research analysis workflows

## Initial MVP
- Clinical question search
- Paper/case/guideline aggregation
- AI-based summary
- Filtering by evidence source
