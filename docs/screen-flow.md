## 1. 목적

본 문서는 MVP 기준 화면 흐름을 정의한다.

## 2. 주요 화면

### 2.1 메인 화면

- 임상 질문 입력창
- 최근 검색 예시
- 주요 필터 바로가기

### 2.2 검색 결과 화면

- 검색어 표시
- AI 요약 영역
- 필터 영역
- 검색 결과 리스트
  - 논문
  - 증례
  - 가이드라인
  - 발표 자료

### 2.3 결과 상세 화면

- 자료 제목
- 출처 유형
- 국내/해외 구분
- 발행일
- 원문 링크
- 요약 정보

## 3. 사용자 흐름

1. 사용자가 임상 질문을 입력한다.
2. 검색 버튼을 누른다.
3. 시스템이 관련 자료를 통합 검색한다.
4. AI 요약과 검색 결과를 함께 보여준다.
5. 사용자는 필터를 적용해 결과를 좁힌다.
6. 사용자는 상세 화면에서 원문과 핵심 정보를 확인한다.
