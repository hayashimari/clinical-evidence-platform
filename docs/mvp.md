# MVP 정의서

## 1. 목적
본 프로젝트의 첫 번째 목표는 한국 의료 환경에 맞는 임상 근거 탐색 플랫폼의 최소 기능 제품(MVP)을 구현하는 것이다.

## 2. MVP 대상 사용자
- 전공의
- 의대생
- 초기 경력 전문의

## 3. 해결하려는 문제
- 임상 질문이 생겼을 때 근거 자료를 빠르게 찾기 어렵다.
- 국내 논문, 증례, 가이드라인이 흩어져 있다.
- 바쁜 의료진은 검색과 정리에 많은 시간을 쓰기 어렵다.

## 4. MVP 핵심 기능
- 임상 질문 검색
- 국내 논문 / 국내 증례 / 국내 가이드라인 우선 검색
- 검색 결과 통합 조회
- AI 기반 핵심 요약
- 국내/해외 자료 구분 표시
- 희귀질환 확장 검색 필터

## 5. MVP 제외 범위
- 커뮤니티
- 댓글 / 토론
- 의사 간 직접 소통 기능
- 자료 직접 업로드 기능
- 진단 확정
- 처방 추천
- 자동 논문 작성

## 6. MVP 성공 기준
- 사용자가 임상 질문에 대한 초기 근거를 짧은 시간 안에 찾을 수 있다.
- 국내/해외 자료를 명확히 구분해 볼 수 있다.
- 검색 결과에서 논문, 증례, 가이드라인을 한 번에 확인할 수 있다.
