# Data Policy

## 1. 목적

본 문서는 플랫폼에서 다루는 데이터의 우선순위와 노출 원칙을 정의한다.

## 2. 기본 원칙

- 국내 자료를 우선 노출한다.
- 해외 자료는 보조 근거로 활용한다.
- 출처 유형을 명확히 구분한다.
- 희귀질환은 예외적으로 해외 증례 확장을 허용한다.

## 3. 데이터 우선순위

1. 한국 가이드라인
2. 한국 논문
3. 한국 증례
4. 한국 학회 / 발표 자료
5. 한국 논문에서 인용된 주요 해외 논문
6. 해외 증례 및 해외 참고 자료

## 4. 데이터 표시 원칙

- 국내 자료 / 해외 자료를 구분해서 표시한다.
- 자료 유형을 명시한다.
  - 가이드라인
  - 논문
  - 증례
  - 발표 자료
- 최신성 정보를 함께 표시한다.

## 5. 필터 정책

- 국내 근거만 보기
- 국내 증례 포함 보기
- 국내 논문에서 인용된 해외 논문만 보기
- 희귀질환 확장 검색 보기

## 6. 주의사항

- 해외 자료는 한국 임상 환경에 그대로 적용 가능한 것으로 간주하지 않는다.
- AI 요약은 원문을 대체하지 않는다.
- 출처 없는 요약은 제공하지 않는다.
