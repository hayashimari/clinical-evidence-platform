# Service Rules

## 1. 목적
본 문서는 플랫폼이 제공하는 정보의 범위와 서비스 운영 원칙을 정의한다.

## 2. 서비스 역할
본 플랫폼은 임상 근거 탐색과 연구 지원을 돕는 도구이며, 의료행위를 직접 대체하지 않는다.

## 3. AI 역할
- 검색 질의 보조
- 논문 / 증례 / 가이드라인 요약
- 핵심 포인트 정리
- 자료 간 공통 패턴 정리

## 4. AI 비허용 범위
- 진단 확정
- 처방 지시
- 의료행위 대체
- 근거 없는 추천

## 5. 자료 제공 원칙
- 가능한 경우 원문 출처를 함께 제공한다.
- 국내 자료를 우선 제시한다.
- 해외 자료는 보조 자료로 명시한다.
- 희귀질환은 예외적으로 해외 자료 범위를 확장할 수 있다.

## 6. 사용자 주의 문구
- 본 서비스는 참고용 정보 탐색 도구이다.
- 실제 진료 판단은 의료진의 책임 하에 이루어져야 한다.
- AI 요약 결과는 원문 검토를 대체하지 않는다.
